# 21vipCRM
CRM(Customer Relationship Management) 시스템. <br/>
기획 서비스: CRM을 기반으로 한 고객 서비스(바우처, 예약) 관리 및 사원 관리 프로그램.
------------------------------------

### 백엔드
Java 11<br/>
SpringBoot<br/>
Spring Data JPA<br/>
Querydsl<br/>
Spring Security<br/>
Oracle<br/>
AWS(Light sail)<br/>
Apache Tomcat<br/>
